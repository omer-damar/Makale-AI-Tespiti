##Makale Özetleri Üzerinden Metin Tespiti
 Bu proje Python ve SQLite teknolojileri kullanılarak geliştirilmiştir.

#Açıklama

Projemizde 3000 adet insan tarafından yazılmış(arxiv) ve 3000 adette AI tarafından oluşturulmuş makale örnekleri veri seti toplanmıştır. Bu veri setleri gerekli veri temizleme aşamalarından geçirilmiştir. 3 farklı Machine Learning (ML) algoritması ile verilerimize uygun şekilde model eğitilmiştir. Geliştirilen yazılım eklenen bir makalenin yüzde kaç ihtimal ile AI yada Insan tarafından yazıldığını 3 farklı ML tahminleri ile yüzdelik oranları ile göstermektedir. Hali hazırda projeye ait Yazılım Şartnamesi, Scrum Table, STD test dokümanı, White Box Testi, SonarQube ve Source Monitor sonuçları belgelerini barındırır.




#Çalıştırma

Önce "cd python" yazarak yolumuzu ayarlayalım
